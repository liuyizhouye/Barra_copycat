# 一、CNE5 的总体结构

* **三类因子** ：国家（Country）、行业（GICS 定制的中国行业因子）、风格（10 个风格）。行业采用 GICS 体系并针对中国市场做了颗粒度定制；文档表 3.1/3.2 给出了 32 个行业因子及其与 GICS 代码的映射与权重示例。
* **国家因子** ：在单一国家模型中显式加入“Country”，本质上可解释为 **以市值加权的国家投资组合** （类似 GEM2 里的 World 因子），使行业因子变成“对冲掉市场”的**美元中性**组合（100% 多行业、100% 空国家），便于归因并提升相关性预测的时效性。
* **风格因子列表** （10 个）：Size、Beta、Momentum、Residual Volatility、Non-linear Size、Book-to-Price、Liquidity、Earnings Yield、Growth、Leverage。所有风格因子 **标准化为：估计宇宙的市值加权均值=0，等权标准差=1** ，从而“风格中性”的参照系可比。
* **估计宇宙** ：使用广义的**全部 A 股**来估计模型，强调行业结构的丰富性与解释力。

# 二、暴露（Exposure）与标准化/正交化

* **风格因子标准化** ：如上，cap-weighted mean = 0、equal-weighted std = 1。
* **正交化（Orthogonalization）** ：
* Residual Volatility 对 **Beta 与 Size** 正交，降多重共线性。
* Liquidity 对 **Size** 正交。
* Non-linear Size：先对 Size 标准化、立方，再 **相对 Size 回归权重正交化** ，再 winsorize 和标准化。

# 三、10 个风格因子：定义、算子与计算口径

> 记：所有权重/半衰期/回看窗均以原文为准，下面逐条列出。

## 1) Size

* **定义** ：1.0 · LNCAP，其中 LNCAP =  **公司总市值的自然对数** 。

## 2) Beta

* **定义** ：1.0 · BETA。
* **计算** ：对**超额收益**做时间序列回归（估计窗口 252 交易日，半衰期 63 日），基准是估计宇宙的 **市值加权超额收益** ：

  ( r_t - r_{ft} = \alpha + \beta \cdot R_t + e_t )。

## 3) Momentum

* **定义** ：1.0 · RSTR。
* **计算** ： **滞后 1 个月** （L=21 日）后，取过去 **T=504 日**的超额对数收益的指数加权和（半衰期 126 日）：

  ( \text{RSTR} = \sum_{t-L}^{t-L+T} w_t [\ln(1+r_t)-\ln(1+r_{ft})] )。

## 4) Residual Volatility（残差波动）

* **定义** ：0.74 · DASTD + 0.16 · CMRA + 0.10 · HSIGMA。
* **DASTD（日波动）** ：过去 252 日**日度超额收益**的波动率，半衰期 42 日。
* **CMRA（累计区间）** ：以月为单位（每月 21 日），构造 **T=1..12 月**的累计超额对数收益 (Z(T))，CMRA = (\max Z - \min Z)。
* **HSIGMA（历史 σ）** ：来自 **回归残差**的波动率（252 日窗口，半衰期 63 日）。
* **正交化** ：对 Beta 与 Size 正交。

## 5) Non-linear Size

* **定义** ：1.0 · NLSIZE。
* **计算** ：以**标准化后 Size 暴露**立方；相对 Size 进行 **回归加权正交化** ，再 winsorize 与标准化。

## 6) Book-to-Price（BTOP）

* **定义** ：1.0 · BTOP =  **最新账面普通股权益 / 当前市值** 。

## 7) Liquidity（流动性）

* **定义** ：0.35 · STOM + 0.35 · STOQ + 0.30 · STOA。
* **STOM（一月换手）** ：过去 21 交易日 **日度换手率之和的对数** ，日度换手率 = 当日成交量/流通股数。
* **STOQ（三月平均换手）** ：把逐月 STOM（每月 21 日）做 **log-exp 平均** ，T=3。
* **STOA（十二月平均换手）** ：同上，T=12。
* **正交化** ：对 Size 正交。

## 8) Earnings Yield（盈利收益率）

* **定义** ：0.68 · EPFWD + 0.21 · CETOP + 0.11 · ETOP。
* **EPFWD（预测盈利/价）** ： **未来 12 个月预测盈利 / 当前市值** ；预测盈利为“当年与次年一致预期盈利”的加权均值。
* **CETOP（现金盈利/价）** ： **过去 12 个月现金盈利 / 现价** 。
* **ETOP（追踪盈利/价）** ： **过去 12 个月盈利 / 当前市值** ；过去 12 个月盈利=上个会计年度值 +（本期中报 − 上年同期中报）。

## 9) Growth（成长）

* **定义** ：0.18 · EGRLF + 0.11 · EGRSF + 0.24 · EGRO + 0.47 · SGRO。
* **EGRLF** ：分析师**长期（3–5 年）**盈利增速预测。
* **EGRSF** ：分析师**短期（1 年）**盈利增速预测。
* **EGRO** ：**过去 5 年**每股盈利（EPS）对时间做回归， **斜率/平均 EPS** 。
* **SGRO** ：**过去 5 年**每股销售额对时间做回归， **斜率/平均销售额** 。

## 10) Leverage（杠杆）

* **定义** ：0.38 · MLEV + 0.35 · DTOA + 0.27 · BLEV。
* **MLEV（市值杠杆）** ：( (\text{ME}+\text{PE}+\text{LD})/\text{ME} )。ME=普通股市值；PE=优先股账面；LD=长期负债。
* **DTOA（负债/资产）** ：( \text{TD}/\text{TA} )。TD=总负债；TA=总资产。
* **BLEV（账面杠杆）** ：( (\text{BE}+\text{PE}+\text{LD})/\text{BE} )。BE=普通股账面值。

# 四、因子收益、截面回归与权重

* **截面回归估计因子收益** ：每期用**加权最小二乘**做截面回归

  ( r_n = \sum_k X_{nk} f_k + u_n )，得到因子收益 (f_k) 与个股特异收益 (u_n)。**回归权重 (v_n)** 与**市值平方根**成正比（常见做法以降低小盘噪声）。
* **解释度与 RMS 分解** ：文中给了总 (R^2) 与 **RMS（均方根）收益**的“x-sigma-rho”截面分解：

  ( \text{RMS} = \sum_k \sigma(X_k),\rho(X_k,r),f_k ;+; \sigma(u),\rho(u,r) )。这里 (\sigma(X_k)) 是因子暴露的 RMS 离散度，(\rho) 为截面相关。

# 五、因子协方差矩阵与两类偏差修正

## 1) 优化偏差调整（Optimization Bias Adjustment, OBA）

* **问题** ：样本协方差的**特征因子（eigenfactors）**存在系统性偏差：对**低波动**特征因子 **低估风险** 、对**高波动**特征因子 **高估风险** ，导致优化组合的风险被系统性 **低估** 。
* **做法** ：在 USE4 方法笔记框架下，用**蒙特卡洛**估计各特征因子的偏差，并 **直接调节特征因子波动率** ，把修正嵌入 **因子协方差矩阵** ，同时 **保留原始纯因子的含义** 。

## 2) 波动体制调整（Volatility Regime Adjustment, VRA）

* **问题** ：波动具有 **非平稳性** ，历史估计会在**波动上升期**低估、在**波动下降期**高估未来风险。
* **做法** ：基于 **截面偏差统计量** （可理解为当日风险预测偏差的即时度量），取**加权平均**得到体制调整因子，分别**用于因子波动**与**特异风险**的标定。

# 六、特异风险（个股残差风险）模型

* **日频序列法** ：先用日度数据直接从**个股特异收益时间序列**估计特异波动（timely、逐股）。但纯 TS 预测对**高波动股**易 **高估** 、对**低波动股**易 **低估** 。
* **贝叶斯收缩** ：按**市值分位**把股票分成 10 桶，在每个 size 桶内计算特异波动的均值与标准差，把个股预测 **向桶内均值收缩** ，离均值越远收缩强度越大。
* **同样应用 VRA** ：对特异风险也做 **波动体制调整** （同上）。

# 七、版本与预测期

* **CNE5S / CNE5L / CNE5D** ：三版本 **暴露与因子收益相同** ，差别在**因子协方差与特异风险预测**的响应度；S 更适合**月度**预测、L 更平滑、D 为 **当日** 。

# 八、一些实务口径与稳定性/共线性诊断

* **风格稳定性系数** ：月度相邻期暴露的截面相关，**<0.80** 视为偏低、**>0.90** 较理想。
* **VIF（方差膨胀因子）** ：衡量因子间共线性，**>5** 通常被视为问题信号。（文中表 4.2 注解）
